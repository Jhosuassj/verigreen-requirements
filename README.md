# VeriGreen - Especificación de Requisitos (BDD)

Este repositorio contiene la documentación viva y formal de los requisitos del sistema **VeriGreen**, estructurada en historias de usuario mediante sintaxis Gherkin.

## Estructura de Módulos
* **Autenticación y Perfiles (US01-US05):** Gestión de acceso para consumidores.
* **Administración y Auditoría (US06-US10):** Registro B2B y validación de empresas.
* **Verificación de Productos (US11-US20):** Funcionalidades core del consumidor para escanear, comparar y validar impacto ambiental.
